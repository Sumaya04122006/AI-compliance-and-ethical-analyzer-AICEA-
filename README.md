# AI-compliance-and-ethical-analyzer-AICEA-
AICEA is a high-tech security guard for AI. It sits between you and the chatbot to keep things legal. It uses two "Agents": The Auditor: Scans for bias or illegal ideas (like ageism). The Fixer: Automatically rewrites text to be safe or blocks dangerous requests. AICEA lets companies use AI without breaking the law.
